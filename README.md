# 🛡️ Advanced Hardware ID Spoofer

Мощный инструмент для подмены аппаратных идентификаторов Windows с функцией автоматического резервного копирования.

## 📌 О проекте

**Advanced Hardware ID Spoofer** — это утилита для Windows, которая позволяет изменять системные идентификаторы оборудования. Программа модифицирует ключевые параметры реестра, связанные с идентификацией компьютера, включая Machine GUID, HWID, SMBIOS данные, информацию о видеокарте и идентификаторы томов дисков.
Подойдет для обхода блокировки VAC, Easy Anti-Cheat (EAC).

### 🎯 Основные возможности

| Модуль | Что изменяет |
|--------|--------------|
| **Registry IDs** | Machine GUID, Product ID, HwProfile GUID |
| **HWID** | Имя компьютера, дата установки Windows |
| **SMBIOS** | Производитель, модель, серийный номер, UUID системы |
| **GPU** | Данные видеокарты (40+ моделей NVIDIA и AMD) |
| **Volume ID** | Идентификаторы всех дисковых томов |
| **Автоматический бэкап** | Создание резервной копии всех изменяемых параметров |
| **Очистка следов** | Временные файлы, Prefetch, журналы событий |

### 🎮 Поддерживаемые видеокарты

- **NVIDIA:** GTX 1050 → RTX 4090 (все серии 10, 16, 20, 30, 40)
- **AMD:** Radeon RX 6600 → RX 7900 XTX
